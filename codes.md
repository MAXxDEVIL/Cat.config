
## vsCode to Github

##### username
``` 
git config --global user.name "Your Name"
```
##### gemail
```
git config --global user.email "your.email@example.com"
```

##### nvme
```
sudo mkfs.btrfs -f -L game_one /dev/nvmeName
```
##### hdd
```
sudo mkfs.btrfs -f -L HDD_one /dev/sda
```
